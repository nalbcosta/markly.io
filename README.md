# Markly.io | 🇵🇹/🇧🇷

Gerador de README.md para perfil do GitHub com preview em tempo real, templates organizados e exportação rápida em Markdown.

## Sobre o projeto

O **Markly.io** é uma ferramenta para criar READMEs de perfil do GitHub de forma simples, rápida e profissional.

A aplicação permite montar um perfil mais organizado, com seções personalizáveis, preview em tempo real e exportação em Markdown.

## Funcionalidades

- Geração de README para perfil do GitHub
- Preview em tempo real
- Templates personalizáveis
- Exportação em Markdown
- Interface moderna e responsiva

## Tecnologias utilizadas

- Next.js
- TypeScript
- Laravel
- Tailwind CSS
- Markdown

## Como funciona

1. Preencha as informações do seu perfil.
2. Personalize as seções desejadas.
3. Visualize o resultado em tempo real.
4. Copie ou baixe o README gerado.

## Seções suportadas

- Apresentação pessoal
- Sobre mim
- Stack principal
- Projetos em destaque
- GitHub Stats
- Links sociais
- Contato

## Objetivo

Facilitar a criação de um profile README bonito, organizado e pronto para destacar melhor a experiência do desenvolvedor no GitHub.

# Markly.io | 🇺🇸

A GitHub profile README generator with live preview, organized templates, and fast Markdown export.

## About the project

**Markly.io** is a tool built to help developers create GitHub profile READMEs in a simple, fast, and professional way.

The application allows users to build a more polished profile with customizable sections, live preview, and Markdown export.

## Features

- GitHub profile README generation
- Live preview
- Customizable templates
- Markdown export
- Modern and responsive interface

## Technologies used

- Next.js
- TypeScript
- Laravel
- Tailwind CSS
- Markdown

## How it works

1. Fill in your profile information.
2. Customize the sections you want.
3. Preview the result in real time.
4. Copy or download the generated README.

## Supported sections

- Personal introduction
- About me
- Main stack
- Featured projects
- GitHub stats
- Social links
- Contact

## Goal

To make it easier to create a clean, organized, and professional GitHub profile README that highlights a developer’s experience in a better way.
